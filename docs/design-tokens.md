# 视觉设计规范 — 色彩系统

基于 bfe-ui / coral design 色彩体系。所有 UI 修改、地图样式、组件颜色均参照此文档。

---

## 主色

| 名称   | 色值      | 用途                     |
|--------|-----------|--------------------------|
| 中行红 | `#B6002A` | 品牌主色，点聚合默认色   |
| 主色   | `#D21F3E` | 按钮、高亮、强调元素     |

---

## 辅助色

| 名称    | 色值      | 用途           |
|---------|-----------|----------------|
| Info    | `#2196F3` | 信息提示       |
| Success | `#4CAF50` | 成功、正常状态 |
| Warning | `#E19600` | 警告、待处理   |
| Danger  | `#F44336` | 危险、错误     |

---

## 中性色

从深到浅，用于文本、背景、边框，通过层次区分视觉结构。

| 名称              | 色值      | 典型用途               |
|-------------------|-----------|------------------------|
| Black             | `#1F2D3D` | 主文本、深色背景       |
| Light Black       | `#324157` | 次级文本、侧边栏       |
| Extra Light Black | `#48576A` | 三级文本、禁用前景     |
| Silver            | `#8391A5` | 占位符、辅助说明       |
| Light Silver      | `#97A8BE` | 边框、分割线           |
| Extra Light Silver| `#BEC8D0` | 浅色边框               |
| Gray              | `#D1DBE5` | 表格线、输入框边框     |
| Light Gray        | `#E4E8F1` | 卡片背景、hover 背景   |
| Extra Light Gray  | `#EEF1F6` | 页面背景               |
| Dark White        | `#FEFBFB` | 内容区底色             |
| White             | `#FFFFFF` | 纯白背景               |

---

## 使用原则

- 地图覆盖物**主题色**优先用 `#B6002A`（中行红），与品牌一致
- 信息分层：正文用 `#1F2D3D`，次要信息用 `#8391A5`，禁用用 `#BEC8D0`
- 背景层次：内容区 `#FFFFFF` → 区块 `#EEF1F6` → hover `#E4E8F1`
- 地图 label 深色背景建议取 `#1F2D3D` 或带透明度的 `rgba(31, 45, 61, 0.88)`
